# Trending Now Dashboard

Live Demo: https://trending-now-dashboard-lqxsbj98m8rxfm8qhdfnvs.streamlit.app/

This dashboard tracks trending content from Reddit, Hacker News, and GitHub.

## Features
- Real-time data from 3 platforms
- Sentiment analysis
- Keyword extraction
- Interactive visualizations

## Tech Stack
- Python
- Streamlit
- Plotly
- Pandas
- VADER Sentiment Analysis
